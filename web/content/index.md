## Setup

:::card title="Installation Guide"
In order to use this reference properly, we recommend you install the Orbit CLI. That way you can make full use of our static site generation and the live reloading tooling of your CLI.

Of course you can also call the Orbit compiler directly, without using the generated client.

#### Documentation Flavour

Select which flavour for generating code examples below:

- **Orbit CLI Client** `enabled`
- **Orbit Node Library** `enable`
:::

:::card title="Quick CLI Setup"
```text
Install the Orbit CLI globally
npm  pnpm  yarn

npm install -g orbit-md
```
:::
